# 📌 Project Overview

The **Amazon India Product Performance Dashboard** is an interactive Business Intelligence (BI) project developed in **Microsoft Power BI**. The dashboard analyzes Amazon India product data to uncover valuable insights into product pricing, customer ratings, reviews, discount strategies, Prime eligibility, bestseller performance, and product availability.

The project demonstrates end-to-end data analysis, including data cleaning, transformation, DAX calculations, and interactive dashboard design.

---

# 🎯 Project Objective

The primary objective of this project is to:

- Analyze Amazon India product data.
- Identify pricing trends across different categories.
- Evaluate customer ratings and review patterns.
- Compare discount strategies between product categories.
- Analyze Prime eligibility and Bestseller products.
- Build an interactive dashboard for business decision-making.
